# Roadmap

This document outlines the planned trajectory for Rapide.

## Released

### v2.0.0: Interactive TUI 🗿🚀
- Full terminal UI with [Charm Bubble Tea](https://github.com/charmbracelet/bubbletea) and [Lipgloss](https://github.com/charmbracelet/lipgloss)
- Interactive list view, real-time filtering, entry creation and editing

### v2.5.0: Git Sync 🌉🚀
- `rapide sync` for push/pull to private Git repos
- Optional autosync on every write
- Setup wizard: `rapide sync --setup`

### v2.6.0: Pinning 📌🚀
- `rapide pin <id>` to keep critical entries at the top

### v2.6.1: Polish & Organization 🛠️🚀
- TUI config editor (`c` hotkey)
- Clipboard/paste support
- Dynamic collection column width

### v2.7.0: Documentation & Onboarding 📖 *(in progress)*
- `rapide init` interactive setup wizard
- TUI help overlay (`?` hotkey)
- VitePress documentation site on GitHub Pages

## Planned

### v3.0.0: Rapide MCP (Model Context Protocol)
- MCP server within the binary
- Contextual search for AI agents
- Automated logging from AI tools
- Expose CLI commands as MCP tools
- Privacy-first: local MCP server

---

*Inspired by the philosophy of Rapid Logging and the aesthetics of the Charm toolchain.*
