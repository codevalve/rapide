# Security Policy

## Supported Versions

We actively provide security updates for the following versions of Rapide:

| Version | Supported          |
| ------- | ------------------ |
| 2.x.x   | :white_check_mark: |
| < 2.0.0 | :x:                |

## Reporting a Vulnerability

If you discover a potential security vulnerability in Rapide, please do not open a public issue. Instead, please report it privately to ensure the safety of our users.

Please send an email to **john.lovell@codevalve.com** with the following details:

- A description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact.

We will acknowledge your report within 48 hours and provide a timeline for a resolution. We appreciate your help in keeping Rapide secure!

---

*Rapide is a local-first application. While we strive for maximum security, always ensure your journal files and Git remotes are appropriately protected.*
